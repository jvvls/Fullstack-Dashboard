## Motivação
Indicadores de inflação são fundamentais para análises econômicas e
sociais. No entanto, dados públicos frequentemente são disponibilizados
em formatos pouco amigáveis para análise e visualização.

## Objetivo
Construir um pipeline simples e reprodutível para transformar dados
brutos do IPCA em visualizações interativas.

## Escopo
- Dados do IPCA para o Brasil e regiões
- Tratamento e padronização dos dados
- API para acesso aos dados processados
- Visualização interativa com D3.js
